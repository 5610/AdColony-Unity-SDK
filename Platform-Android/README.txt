Getting Started with AdColony:
----------------------------------
First time and returning users should review the AdColony SDK Quick Start Guide. 
This file is available as a PDF in the "Documentation" folder and contains everything 
you need to start integrating AdColony into your app quickly.   

1.9.11 Summary Change Log:
----------------------------------
1. Removed GET_TASKS permission requirement.2. Added READ_PHONE_STATE permission requirement.3. AdColony plugin will now call AdColony.pause() from the Unity's onApplicationPause() and AdColony.resume(this) when Unity resumes.

1.9.9 Summary Change Log:
----------------------------------
Details can also be found in the AdColony Quick Start Unity Android.pdf under "Changes"
-Fixed an issue with V4VC post-popups.
-Added support for multiple scene projects.
-Removed dependency on 'Main Camera' object.
-Updated with AdColony 1.9.9 Android SDK (Which includes several critical fixes)

1.9.7 Summary Change Log:
----------------------------------
IMPORTANT: November Update added Unity Plugins. If using Unity plugin, please use the Library and Documentation under Plugins folder.
Information on setting up and using the plug-in can be found in the header comment of Assets/Plugins/AdColony.cs and AdColony Unity Quick Start guide.
